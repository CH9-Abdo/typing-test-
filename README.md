# PythonType

A minimalist typing test application built with Python (PyQt5 or Pygame), inspired by Monkeytype.

## Features
- **Clean Interface**: Dark mode with distraction-free typing area.
- **Real-time Feedback**: Instant character validation (White: Correct, Red: Incorrect).
- **Live Metrics**: Words Per Minute (WPM) and Accuracy tracking.
- **Game Modes**: Standard 30-second sprint.
- **Quick Restart**: Press `Tab` at any time to reset.

## Requirements
- Python 3.x
- PyQt5 (for `main.py`) or pygame (for `main_pygame.py`)

## Installation
```bash
pip install PyQt5    # for PyQt5 version
pip install pygame   # for Pygame version
```

## How to Run

**PyQt5 version (default):**
```bash
pip install PyQt5
python main.py
```

**Pygame version (same app, different UI):**
```bash
pip install pygame
python main_pygame.py
```
